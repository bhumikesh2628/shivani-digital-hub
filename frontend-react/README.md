# CSV Shivani - Frontend (React + Vite)

Frontend for CSV Shivani Digital & Business Services website.

## Tech Stack

- **React 19** - UI library
- **Vite** - Build tool and dev server
- **CSS** - Custom styling with responsive design

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open browser to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features

- Responsive design for all devices
- Contact form with validation
- Smooth scroll navigation
- Service showcase
- Testimonials section
- Social media integration

## Contact Form

The contact form sends messages directly via WhatsApp to your phone number. No backend or database required.

To change the phone number, edit `src/components/Contact.jsx` and update the `phoneNumber` variable.
